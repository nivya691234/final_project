🚀 HOW TO USE THE NEW SYSTEM
═══════════════════════════════════════════════════════════════════════════════

ICONS ARE AUTOMATIC:
  The icons.js script automatically replaces all emoji on page load.
  You don't need to do anything - just make sure base.html is updated
  (which it already is).

TO USE ANIMATIONS IN YOUR CODE:

  // Update KPI with animation
  setText('kpi-cpu', 45.5);

  // Update progress bar with animation
  setBar('bar-ram', 67);

  // Show notification
  showToast('Process updated!', 'success');

  // Highlight new data
  highlightNewData(document.getElementById('proc-tbody'));

  // Manual animation
  animateValueChange(element, newValue, { duration: 800 });
